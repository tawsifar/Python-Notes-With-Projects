# Python Tutorial Notes

A structured Python learning series written by a CSE student at CUET.
Covers 60+ topics from variables and data types all the way to OOP,
PyQt5 GUI development, file handling, multithreading, and more.
Each file is a standalone topic with clean tutorial comments explaining
every concept as you go.

## Topics Covered
- Python basics: variables, data types, typecasting, input/output
- Control flow: if-else, loops, match-case
- Functions: args, kwargs, lambda, decorators, generators
- OOP: classes, inheritance, polymorphism, magic methods
- File handling: txt, json, csv read/write
- Modules: random, datetime, math, os
- GUI: PyQt5 widgets, layouts, buttons, stylesheets
- Multithreading, API requests, exception handling

## Projects Included
20 beginner to intermediate projects including:
Calculator, Banking Program, Slot Machine, Hangman,
Encryption Program, Weather App, Digital Clock, Stopwatch and more.

## Stack
Python 3.10+ | PyQt5 | requests | pygame
